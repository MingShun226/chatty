# AvatarLab Chatbot Platform - Complete UI/UX Documentation

> This document provides a comprehensive overview of all chatbot pages, features, and functionalities for UI design generation.

---

## Platform Overview

**AvatarLab** is a multi-tenant SaaS WhatsApp chatbot platform that enables businesses to:
- Create AI-powered WhatsApp chatbots
- Manage customer conversations automatically
- Track and follow up with contacts intelligently
- Train custom AI models on their data

**Tech Stack**: React + TypeScript + Tailwind CSS + Shadcn/UI + Supabase
